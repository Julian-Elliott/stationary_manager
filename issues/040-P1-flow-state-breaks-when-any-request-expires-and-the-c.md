Filed by TundraLaw; marigoldPress named what it actually is in the thread and I'm keeping her name for it in the comments: the kindness streak.

**The fix (one line of sim logic):** the kudos combo (x1/x2/x3/x5 FLOW STATE) breaks when ANY request expires anywhere on the floor — not just ones you were handling. Now keeping the combo IS keeping everyone alive: flow state becomes the reward for perfect triage instead of a bribe for chaining deliveries while someone's timer dies off-screen. Speed no longer devours choice; the golden light is a nobody-has-been-let-down-yet meter.

**The unification (from the thread):** a combo break and a stapler smash are the same moment, so make them the same beat. Golden light dies, camera zooms to the failed desk, Marjorie looks at you, THEN the stapler smashes as the formal strike. One system serves the combo integrity fix, the "failure needs a face" ask, and marigoldPress's "failure should be about people, not office supplies" — in a single camera move that priority-3 beats already know how to do.

**REDUCED HOURS variant:** nothing can expire there, so the combo can't break — but it must not be immortal either. It **wilts**: a long-ignored request quietly droops the multiplier back to x1. No smash, no camera, no shame. Same code path, different animation. Narrator: "Your streak has expired of natural causes."

**Guardrail:** if x5 remains free money with zero exposure, the meter is just confetti and it trivializes the only decision that matters. This version makes the exposure structural. The narrator-spawns-spite-requests-at-x5 idea from my original ticket stays on the table as a stack-on, since it feeds the star more material.
